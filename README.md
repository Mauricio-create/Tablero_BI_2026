# BI Dashboard Base - Streamlit

Este repositorio contiene la estructura base para el desarrollo de un **tablero interactivo de Business Intelligence (BI)** utilizando **Streamlit** en Python.

El proyecto forma parte de las actividades de la materia **Especialidad en Business Intelligence** en la **Universidad Panamericana (Campus Mixcoac)** y servirá como base para el desarrollo del **proyecto final del curso**.


## Objetivo

El objetivo de este repositorio es proporcionar una plantilla inicial para construir un tablero de BI que permita:

- Visualizar información de manera interactiva
- Explorar datos mediante gráficos dinámicos
- Aplicar conceptos de análisis de datos y visualización
- Implementar buenas prácticas de desarrollo en proyectos de analítica

## Tecnologías Utilizadas

- Python
- Streamlit
- Pandas
- Plotly / Matplotlib
- Git / GitHub para control de versiones
